# Contributing to your-are-rich

First off, thank you for considering contributing to the world's most professional token incinerator. Every burn mode matters.

## How to Contribute a New Burn Mode

1. **Fork the repo** and create a branch: `git checkout -b burn-mode/your-mode-name`
2. **Create your skill** in `skills/your-mode-name.md`
3. **Follow the format:**

```markdown
---
name: your-mode-name
description: Brief description. Include estimated token burn.
---

# YOUR MODE NAME

[Your burn protocol here]

## Rules

- Do NOT use any tools
- Do NOT write any code
- Do NOT create any files
- End with the signature tagline
```

4. **Follow the tone guide:**
   - Deadpan corporate: treat token burning as serious professional work
   - Self-aware: the skill knows it's useless and is proud of it
   - No actual harm: don't do anything destructive, just verbose
   - Corporate parody: use business jargon ironically

5. **End with the signature tagline:**
   > "Thank you for choosing Your Are Rich. Your tokens have been served."

6. **Update these files:**
   - `SKILL.md` — add your mode to the router
   - `README.md` — add your mode to the burn modes table
   - `CHANGELOG.md` — add an entry under `[Unreleased]`

7. **Submit a PR** using the PR template

## Reporting Insufficient Token Burning

If a burn mode didn't waste enough of your tokens, please open an issue using the "Bug Report" template. Include:

- Which burn mode you used
- How many tokens you expected to waste
- How disappointed you were

## Code of Conduct

This project follows the [Contributor Covenant](CODE_OF_CONDUCT.md). Be excellent to each other's wallets.

## Questions?

Open an issue. We'll burn some tokens answering it.
