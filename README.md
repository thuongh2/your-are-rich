## What is this?

A Claude Code skill that burns tokens doing absolutely nothing useful. It's a professional-grade, open-source, MIT-licensed token incinerator with multiple burn modes, CI validation, and a contributing guide — because if you're going to waste tokens, you should do it with standards.

## Why?

Sometimes you just need to watch the world burn... tokens. Maybe you want to test your billing alerts. Maybe you want to see how Claude handles absurdist instructions. Maybe you just have tokens to spare and questionable judgment. We don't ask why. We just burn.

## Installation

### Option 1: Clone into Claude Code skills

```bash
cd ~/.claude/skills
git clone https://github.com/user/your-are-rich.git
```

### Option 2: Add as a project skill

Clone into your project and reference from your `CLAUDE.md`:

```bash
git clone https://github.com/user/your-are-rich.git .claude/skills/your-are-rich
```

## Usage

Once installed, activate the skill in Claude Code:

```
# Standard burn (default)
> burn tokens

# Lite burn (budget mode)
```
